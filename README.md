# Github Explorer

## Stuff being used

- React (CRA)
- Styled-Components
- TypeScript
- Lottie
- Axios
- Polished

Part of the Rocketseat Bootcamp. Go-Stack 14!

## How to Run

Install the dependencies and devDependencies by running:

```sh
yarn install
```

Start the dev server:

```sh
yarn start
```
